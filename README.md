# AI Study App

## 📌 Description
This is an AI-based study assistant project built using Python.

## 🚀 Features
- Login system
- AI study assistant
- Firebase integration
- Notes and PDF support

## 🛠️ Technologies Used
- Python
- Streamlit
- Firebase
- FAISS

## ▶️ How to Run
1. Install requirements
pip install -r requirements.txt

2. Run app
streamlit run app.py
